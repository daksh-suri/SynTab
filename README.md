# Tab Manager Chrome Extension

A simple Chrome extension for managing browser tabs.

## How To Install

1. Download the extension ZIP file.
2. Extract/unzip the downloaded file.
3. Open Chrome.
4. Go to:

```txt
chrome://extensions
```

5. Turn on Developer mode.
6. Click Load unpacked.
7. Select the extracted extension folder.
8. Pin the extension from the Chrome toolbar.
9. Click the extension icon to start using it.

## Important

Do not select the ZIP file directly in Chrome.

You must unzip it first, then select the extracted folder using Load unpacked.

## Features

- View open tabs grouped by window
- Search tabs by title or URL
- Switch to a tab
- Close one or multiple tabs
- Save tabs as a session
- Restore saved sessions
- Delete saved sessions

## Notes

This extension is distributed as a ready-to-load Chrome extension folder.

No coding setup is required.
